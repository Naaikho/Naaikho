## About

I design and ship web, mobile, desktop, and automation products with a strong focus on clean interfaces, useful workflows, and reliable backends.

My work sits between **development**, **product design**, and **creative tooling**: SaaS platforms, admin dashboards, real-time applications, AI workspaces, music-production utilities, and custom business software.

```txt
Naikho = developer mindset + designer eye + maker energy
```

## Selected work

| Project | What it is | Stack |
|---|---|---|
| **Smart Lock — ECO PBC** | Smart-locker infrastructure for automated vending machines: admin platform, Android app, backend API, stock, sales, monitoring, and machine operations. | Next.js · React Native · Express.js · Prisma · Socket.IO |
| **Infinity Table** | Restaurant reservation SaaS with booking management, schedules, pre-authorizations, customer communication, and public booking flow. | Next.js · TypeScript · Stripe · Day.js · SaaS |
| **U-Academy** | E-learning platform for learners and instructors with progression, quizzes, document library, blog management, and polished frontend. | Nuxt.js · Vue.js · TypeScript · Tailwind CSS |
| **QR Dine** | QR-code restaurant ordering platform with menu browsing, real-time order tracking, online payments, and staff administration. | Next.js · TypeScript · Express.js · Prisma · Socket.IO |
| **SpendLess Daw** | Desktop project manager for FL Studio projects: `.flp` metadata extraction, sample dependency safety, preview, archive, and organization. | TypeScript · Electron · React · FL Studio · Binary parsing |
| **Naura Prospekt** | Long-term personal AI workspace inspired by a “Jarvis” idea, combining conversational AI, notes, voice input, tools, settings, and productivity systems. | Next.js · TypeScript · Express.js · OpenAI · Python |
